# 1. Choisir le projet

## Pour le sujet
- `cd Sujet_web_2024`

## Pour la correction
- `cd Correction_web_2024`

# 2. Installer les packages
1. `cd backend`
2. `npm i`
3. `cd ..`
4. `cd frontend`
5. `npm i`

# 3. Lancer le projet

## Lancer le backend
- `cd backend`
- `npm start`

## Lancer le frontend
- `cd frontend`
- `npm run dev`
